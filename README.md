# IPHAN_DIVGEO
Repositorio de scripts utilizados em em fluxos de processos para análise licenciamento ambiental, tais como rotinas de análise para inclusão de áreas prioritárias e análises manuais SIAP.
## Pastas:
1. Calculo de áreas
   
   Nesta pasta estão incluídas opções de código com a finalidade de automatizar o cálculo de áreas de polígonos em lote.
   
2. Download de dados
   
   Incluídas opções em Python e R para baixar dados atualizados de fontes de dados institucionais, como IBGE e MMA por meio do pacote GeoBr desenvolvido pelo IPEA.
   Pereira, R.H.M.; Barbosa, R.J.; et. all (2026) geobr: Download Official Spatial Data Sets of Brazil. v2.0.0 GitHub repository - https://github.com/ipea/geobr.
